# CompanyName

A Windows utility tool for encoding and patching company name strings inside compiled EXE files.

## Features

- Encode / Decode company name strings
- Patch company code directly into EXE binary
- Edit EXE version resources (Product Name, File Description, Company Name, etc.)
- Rename EXE title bar text
- Supports first-time initialization on clean/unpatched EXE files

## Usage

1. Run `CompanyName.exe`
2. In the **Company Name Changer** panel, drag & drop or browse your target EXE
3. Enter the new company name and click **Rename Company Code**
4. Use the **Text Encryption Tool** to manually encode or decode strings

## Requirements

- Windows 10 / 11 (x64)
- No installation required — standalone executable
